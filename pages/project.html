项目简介

PROJECT BACKGROUND

项目背景
2013年开始，各路大伽纷纷开展外卖业务，形成外卖市场的“百团大战”，经过多年的合并整合，目前仅剩美团、百度、饿了么等规模较大的外卖订餐平台。成熟巨大的外卖市场仍存在着非常大的发展空间；几大外卖平台仅仅只是拥有网上订餐配送系统，运营上均是采用商家入驻的模式。据不完全统计，在中国每天外卖订单量超2000万单，90%的订单来自这三大平台。
由于三大平台运营模式相同，导致消费者无论在哪个平台订外卖得到的商品都是一样的----来自线下商家。从而导致消费者对外卖餐品的乏味，由于外卖又是一种高频次的消费行为，所以“饭店味道”，靠大量味精、香精烹调出的食品实在是很难天天享用。而加盟外卖平台的商家又是良莠不齐的，餐品加工环节和食材质量监管方面均存在很大的盲点和漏洞，一些黑作坊及食品安全问题时有发生，问题及矛盾越来越突出。2017年两会上，外卖的食品安全问题作为议题被提上了议程，为解决上述问题，“家庭式中心厨房+外卖”模式应运而生。

PROJECT BACKGROUND

项目优势

家庭式中心厨房相对餐饮行业实体店家优势突出，主要有以下几个方面：

1、中心厨房80%的空间为炉灶灶位，大大提高了出菜量；每人只做一道菜，大大提高了出菜效率。做到餐饮实体店的利润最大化。

2、“家庭式”理念贯穿整个项目，弥补了整个餐饮市场的空白，从菜品上做到了独一无二，专业厨师做的菜到处都是，而家庭味道的菜只有我们这才有。

3、严格透明的监管，解决了食品安全这个老大难问题，更好的留住用户，高强度的粘性必然带动高频次的消费。候么外卖经营模式，在互联网餐饮这个领域里是个创举，O2O+B2C模式更有着得天独厚的独特优势。

4、相较于其他外卖平台所采取的店家入驻模式，其菜品盈利点在20%之内的抽成，而我们则把菜品的利润提高到了45%，而且还享受到了“候么外卖APP”的商业价值。

5、中心厨房把单菜品的利润提高到了45%以上，面对某团等三大平台的竞争我们更有优势，同样做5折的活动，三大平台至少贴3折，而候么外卖最多是不赚钱，在活动的力度和周期上有着得天独厚的优势。

6、采用招商加盟制度，“5灶位即可代理，15平方的面积5万元即可开店”等一系列的低门槛加盟政策，可以做到迅速发展门店；代理费的收取，完成“自我供血，自主盈利”的发展要求。即使发展期及后期没有A轮等大的融资介入，也可以完成快速布局及持续良性发展的战略要求。

7、门店布局，统一门头，自主配送，服装统一，都有着超强的市场宣传作用，而且是一条免费的宣传途径。省去高昂广告费的同时，提高了宣传力度。

PROJECT BACKGROUND

项目前景
从行业上分析：
候么外卖是一家互联网企业，而互联网行业又是目前中国最具潜力与未来的行业；其次，候么外卖又是一家连锁餐饮企业，而餐饮业又是传统行业中比较稳定且赚钱的行业。互联网和餐饮两个行业相互配合、相互依托、相互支持，前景不言而喻。
从市场空间上分析：
随着时代的发展与进步，外卖已逐渐成为主流的餐饮消费模式，目前全国外卖日订单量超过了2000万单，这个数字还在以每年 17%的速度增加，是一个年产值千亿级的市场，前景不可限量。
从用户需求上分析：
我们的“家庭式”和“全民监管”方式，使得我们的用户粘性很高，我们可以不作为主流外卖模式，但是我们可以弥补外卖市场上的空白，与主流外卖模式形成互补。互联网最缺的就是粘性，有了高粘性的大量用户，作为互联网行业成功上市的前景巨大。
从政策上分析：
食品安全问题一直是一个老大难的问题，今年两会上对外卖行业的食品安全问题也作为议题被提上了议程，而我们的模式及操作办法直接且有效的解决了会上提出的 9成难点。所以，我们今天的模式与布局是整个行业未来的政策性导向，先见之明和提前布局决定着前景有着旺盛的生命力。
从经济热点上分析：
目前比较热门的两个问题“共享经济”和“最后一公里”两大课题。我们中心厨房中的灶位共享，对于 B端的“家庭煮妇（夫）”而言，为他们提供了就业与创业的机会。通过大量的市场经验与调研分析，我们可以精准的锁定用户群体，使得中心厨房的选址更具针对性，把配送范围缩小到了 300米至一公里范围内。综上所述候么外卖的前景巨大。

PROJECT BACKGROUND

盈利分析
盈利模式
候么外卖的盈利模式很简单，从本质上讲就是一点：营业额收入。候么外卖采取直营+代理（连锁加盟）的运营模式，无论是直营还是代理，盈利点最终都来自于营业额。直营中心厨房，营业额的所有利润直接为公司收益；而代理中心厨房，公司除前期收取一次性平台使用费外（2000元/灶位），还要每日收取营业额5%的平台管理费。
中心厨房盈利分析

中心厨房建设费用为每灶位3000元（包建模块）计算，另加厨房自建费用不等。

一线城市以北京为例
一线城市以北京为例分析（100平方场地建设30灶位）

投资成本：北京二环内房租平均成本在12元/㎡，以最高值15元/㎡计算。 房租成本为：15元/㎡ * 100㎡ * 365天 = 547500 元 建设成本：30位 * 3000 = 90000 元 固定投资金额合计 =总投资650000元 /30/30/13/365=15%总营业额成本 运营成本：营业额的55%为运营成本（人工、原材料、水电等等，不含房租） 利润：一个标准位平均单日出菜量为30份，炒菜和烧菜比例是2:1，平均菜 价为13元/份。 30灶位 * 30份 * 13元 = 11700元 * 45%纯利润 = 5265 元纯利润 5265 * 365天 = 1，921，725元 – 650，000 = 1，271，725元纯利润 650,000 / 5265 = 123天收回成本

三线城市以徐州为例分析（100平方场地建设30灶位）

投资成本：徐州市内中心房租平均成本在5元/㎡/天，以最高值8元/㎡计算。 房租成本为：8元/㎡ * 100㎡ * 365天 = 292000 元 建设成本：30位 * 3000 = 90000 元 固定投资金额合计 =总投资382000元 运营成本：营业额的55%为运营成本（人工、原材料、水电等等，不含房租） 利润：一个标准位平均单日出菜量为30份，炒菜和烧菜比例是2:1，平均菜 价为10元/份。 30灶位 * 30份 * 10元 = 9000元 * 45%纯利润 = 4050 元纯利润 4050 * 365天 = 1，478，250元 – 382,000 = 1,158,250元纯利润 382000 / 4050 = 94天收回成本
